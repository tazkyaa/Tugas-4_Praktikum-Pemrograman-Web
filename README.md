Pada tugas ini, saya diminta membuat sebuah web sederhana untuk pembagian Tim Merah dan Tim Biru.
Tugas utamanya adalah menghubungkan tampilan web (HTML/CSS) dengan database MySQL, sehingga data peserta dapat melakukan proses CRUD (Create, Read, Update, Delete).

Saya menggunakan template yang sudah disediakan, lalu mengisi bagian backend-nya menggunakan PHP. Semua halaman selain login dan register harus menggunakan session, sehingga hanya user yang login yang bisa mengakses fitur utama.
